## 검증 가능 범위

| 게임 없이 검증 (테스트벤치) | 게임 필요 |
|---|---|
| LLM 응답 action 정확성 | GenStep 패치 실제 맵 반영 |
| params 구조 유효성 | Map Preview 미리보기 갱신 |
| 타일 조건부 응답 | UI 렌더링 / 스크롤 |
| 파라미터 범위 클램핑 | 성능 (프레임 드롭) |
| 한국어 처리 | 모드 호환성 |
| mutators 배열 포함 여부 | TileMutator 실제 적용 |
| coast_direction/rock_count/ore_density 파라미터 | 석재/광석/해안 실제 반영 |
